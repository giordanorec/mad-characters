# mad — 100 Sheets (analogia × estética)

Cada **sheet** = uma **analogia** (o que o role vira) + uma **estética** (como é
desenhado). Cada sheet gera 100 personagens (1 por role). 100 sheets × 100 roles
= 10.000 personagens.

Formato: `NN. Analogia — Estética · vibe`
